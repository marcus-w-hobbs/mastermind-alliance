# Mastermind Alliance

Open-source implementation of the Mastermind Alliance concept — AI personas engaging in philosophical dialogue using the Backrooms architecture.

## Features

- **Mastermind Tab** — Multi-persona roundtable discussions
- **Backrooms Tab** — Stream-of-consciousness AI dialogues  
- **Ablation Studies** — Research on prompt engineering effectiveness

## Quick Start

```bash
# Install
npm install

# Add your API key
cp .env.example .env.local
# Edit .env.local with your ANTHROPIC_API_KEY

# Run
npm run dev
```

## No Authentication Required

This is a focused, open-source version. Add your own API keys and go.

## Research

See `experiments/ablation/` for prompt ablation studies demonstrating which system prompt components are load-bearing vs decorative.
